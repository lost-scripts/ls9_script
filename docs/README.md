	<p>There are actually two ways for installing custom scripts in MohoÂ® and they apply the same to this kind of shareed resources container, so (as usually) you can choose the one that better suit your needs...</p>

			<td style="border: 1px solid #4E4F4A; padding: 8px;">

				<li>Unzip the downloaded file.</li>

				<li>Drag & Drop (or Copy & Paste) all of them into the <b><i>Scripts</i></b> folder of <u>your <b><i>Custom Content Folder</i></b></u> (overwrite if prompted).</li>
				<li>Restart MohoÂ® or press "Alt + Shift + Ctrl + L" to <i>Reload Tools And Brushes</i> and that should be all.</li>
			</ol>
